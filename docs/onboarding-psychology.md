# Onboarding Psychology

## Principles
- Build trust before asking for sensitive data
- Use progressive disclosure
- Reduce uncertainty before commitment
- Avoid asking too much too early
- Explain why data is needed
- Use reassurance before high-friction steps

## Common Friction Points
- Asking for NRIC before explaining why
- Asking income before eligibility logic is clear
- Manual review warning without next-step explanation
- Long forms without progress indication
- Technical language before user motivation is established
